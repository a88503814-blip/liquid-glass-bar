Everything you need

A status bar that
actually does something

Every widget is live, accurate and instant. No polling jank, no resource drain.

💻

System Monitor

Real-time CPU and RAM using the same Windows kernel counters as Task Manager. No child processes, no fake readings.

🎵

Media Controls

Detects whatever is playing — Spotify, YouTube, Chrome, Edge. Click to get play/pause/skip controls with album art.

🔔

Notifications

Clean dismissable popup cards replace Windows toast notifications. Works with every app including Chrome and Discord.

🌐

Network & Ping

Live Wi-Fi or Ethernet detection with real TCP ping. Colour-coded: green below 50ms, yellow to 120ms, red above.

☁️

Weather

Live temperature and conditions — no API key needed. Auto-detects your city by IP or set a custom location.

🪟

Virtual Desktops

Click any desktop number to switch instantly. Also detects and works with GlazeWM and Komorebi tiling managers.

🍅

Pomodoro Timer

Built-in focus timer. Left-click to start/pause, right-click to reset. Notifies you when focus or break ends.

🔋

Battery Saver

Battery widget with charging detection. Below 20% the bar automatically kills all animations to save power.

🔄

Auto-Updates

When a new version is released, a pill appears in the bar. Click once to download and install in the background.

Get started

Up and running in 2 minutes

No account. No subscription. No configuration required. Just install and go.

1

Download

Click the download button. You get a single .exe installer — nothing else to install.

2

Install

Run the installer. It places the app in Program Files, creates shortcuts, and adds itself to Windows startup.

3

Use it

The bar appears at the top of your screen. Maximised windows stop at the bar edge — no overlap, ever.

4

Auto-updates

Future releases show a glowing pill in the bar. Click it → downloads in background → click again to install.
